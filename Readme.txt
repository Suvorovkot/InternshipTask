Восстановление базы данных
1) Запустить PgAdmin
2) Создать новую базу c названием Users (пользователь postgres, пароль postgres)
3) Щелкнуть праой кнопкой мыши по названию базы и выбрать RESTORE
4) Указать путь до файла, лежащего в папке проекта
.../InternshipTask/src/main/resources/UsersDB

Само приложение запускается на localhost:5000 нажатием кнопки Run в IntelliJ Idea или же командой sbt run
